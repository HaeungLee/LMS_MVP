## 🎯 핵심 목표
**"매일매일 진행되는 수업 진도에 맞춰 AI가 문제를 추가 및 채점, 피드백"**

---

## 📋 구현 로드맵

### 🔥 **Phase 1: AI 피드백 활성화** (오늘 완료 목표)
**예상 시간**: 4-6시간

#### 1.1 OpenRouter API 실제 연동
- [ ] `backend/app/services/llm_providers.py`에서 템플릿 → 실제 API 호출
- [ ] 문제별 맞춤 피드백 프롬프트 설계
- [ ] 비용 절약용 캐싱 시스템 강화
- [ ] 에러 핸들링 및 폴백 메커니즘

#### 1.2 채점 AI 고도화
- [ ] 단순 정답 매칭 → AI 기반 의미 분석 채점
- [ ] 부분 점수 시스템 (0점, 0.5점, 1점)
- [ ] 오답 패턴 분석 및 맞춤 힌트 생성

#### 1.3 실시간 피드백 시스템
```
기존: 제출 → 정답 확인 → 템플릿 피드백
개선: 제출 → AI 채점 → 개인화 피드백 → 추가 학습 제안
```

---

### 🚀 **Phase 2: AI 문제 생성 자동화** (내일 완료 목표)
**예상 시간**: 6-8시간

#### 2.1 수업 진도 추적 시스템
- [ ] 교사용 진도 관리 대시보드
- [ ] 일일 학습 목표 설정 (토픽/난이도/문제 수)
- [ ] 진도율 기반 자동 문제 생성 트리거

#### 2.2 AI 문제 생성 엔진
```python
# 예시 워크플로우
daily_topic = "파이썬 리스트"
difficulty_level = "중급"
student_weakness = ["슬라이싱", "메서드 활용"]

generated_questions = ai_generate_questions(
    topic=daily_topic,
    difficulty=difficulty_level,
    focus_areas=student_weakness,
    count=5
)
```

#### 2.3 문제 품질 관리
- [ ] AI 생성 문제 자동 검증 시스템
- [ ] 교사 승인/수정 워크플로우
- [ ] 문제 난이도 자동 조정 알고리즘

---

#### 3.1 학습자 프로파일링

- [ ] 학습 속도 맞춤 조정

#### 3.2 적응형 문제 출제
- [ ] 실시간 난이도 조정
- [ ] 개인별 맞춤 문제 큐레이션
- [ ] 복습 타이밍 최적화
