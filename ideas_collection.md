## 💡 **기존 개선사항 (추가하면 좋을점 파일 기반)**

### 🎯 단기 개선사항 (1-2주)
1. **실시간 AI 피드백 통합**
   - OpenRouter API 실제 연동 (현재는 템플릿 사용)
   - 문제별 맞춤 피드백 및 힌트(정답X) 생성
   - 피드백 캐싱으로 성능 최적화

2. **문제 카테고리 확장**
   - 현재: Python 기초만 지원
   - 추가: JavaScript, HTML/CSS, SQL, 알고리즘 등
   - 카테고리별 문제 뱅크 구축

3. **학습 진도 추적**
   - 문제별 정답률 통계
   - 취약 영역 분석 대시보드
   - 학습 추천 시스템

### 🏢 중기 기능 (1-2개월)
4. **사용자 인증 및 권한 관리**
   - 학생/교사 계정 분리
   - JWT 기반 인증 시스템
   - 개별 학습 이력 저장

5. **교사용 관리 패널**
   - 문제 출제 인터페이스 (CRUD)
   - 학생 성적 관리 대시보드
   - 반별/개별 학습 현황 모니터링
   - 문제 통계 (정답률, 난이도 분석)

6. **고급 퀴즈 기능**
   - 시간 제한별 모드 (5분, 10분, 무제한)
   - 랜덤 문제 생성 알고리즘
   - 적응형 난이도 조절
