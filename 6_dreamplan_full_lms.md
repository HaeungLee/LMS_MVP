### DreamPlan: 확장형 Full LMS 로드맵

비전: LLM이 개인별 학습 진도/성취/관심사를 반영해 교과서·커리큘럼을 자동 생성/재구성하고, 진로·직무 추천 및 기업 연계까지 지원하는 교육 플랫폼.

---

## Phase A — 강건한 코어(3개월)
목표: 안정적인 코스/평가/영속화 기반 + 기본 AI 피드백

Tasks
- [ ] 도메인 모델 고도화: Course/Module/Lesson/Quiz/Attempt/Gradebook/Question Bank
- [ ] 인증/권한: JWT + RBAC, 멀티테넌시(기관/반)
- [ ] 출제 워크플로: 작성→검토→승인, 버전관리, 통계(정답률/평균시간)
- [ ] AI 피드백 2-Agent(루브릭 기반), 캐시/비용 모니터링, 가드레일
- [ ] 코드 실행 샌드박스(코딩), 리소스 제한/타임아웃, 테스트케이스
- [ ] 대시보드/리포트: 학습/정답률/시간/활동 추세, Export

AC
- [ ] 단일 코스 이수 흐름(E2E) 100% 통과
- [ ] AI 피드백 환각률 감소(내부 샘플셋 기준)
- [ ] 샌드박스 안전성/재현성 보장

---

## Phase B — 개인화/추천(3~6개월)
목표: 학습 데이터 기반 개인화 추천과 적응형 커리큘럼

Tasks
- [ ] 취약영역 분석 고도화(문항/주제/코스 레벨)
- [ ] 학습 경로 추천: 목표/약점/선호 기반 코스/문항 추천
- [ ] LLM 기반 커리큘럼 리라이트: 최근 성과에 맞는 콘텐츠/문항 재배치
- [ ] 힌트 엔진: 단계별 힌트(정답 미노출), 오답 유형별 페르소나 피드백
- [ ] 목표 설정/리마인더/알림(캘린더)

AC
- [ ] 추천 적합도/학습 유지율 향상(AB 테스트)
- [ ] 커리큘럼 자동 조정 후 성취도 상승 지표

---
## Phase C — 직무 역량화/기업 연계(6~12개월)
목표: 실무형 프로젝트/직무 역량 매핑/채용 파이프라인 연계

Tasks
- [ ] 직무 스킬 매트릭스 정의(신입 기준), 코스/문항과 매핑
- [ ] 캡스톤/미니프로젝트 평가 루브릭, 포트폴리오 자동 생성
- [ ] 기업 연계: 과제형 평가/모의 면접/프로필 공유 동의 기반 추천
- [ ] 멘토링/피어 리뷰/팀 과제(협업 학습)

AC
- [ ] 기업 파일럿(채용 연계) KPI 달성
- [ ] 포트폴리오 자동 생성 품질 검증(리뷰어 평가)

---

## Phase D — 학제 확장/표준 연동(12개월+)
목표: K-12/대학/기업교육 전개, 표준/규정 준수, 글로벌화

Tasks
- [ ] K-12 커리큘럼 맵핑: 학년/과목/성취기준 연결
- [ ] SCORM/xAPI/LTI 연동, SSO(교육기관)
- [ ] 접근성/WCAG 준수 강화, i18n 다국어, 모바일/오프라인 모드
- [ ] 정책/컴플라이언스: FERPA/GDPR/CCPA, 데이터 거버넌스

AC
- [ ] 기관 도입 PoC 성공, 성적/콘텐츠 호환 문제 없음
- [ ] 글로벌 지역화(i18n) 검증 지표 충족

---

## 기술 전략(가이드)
- 데이터: OLTP(운영 DB) + OLAP(웨어하우스/리포팅) 분리, ETL 배치
- ML/LLM: 프롬프트 템플릿/루브릭/평가셋 관리, 캐시 키(문항ID+답안해시+루브릭버전)
- 보안/운영: IaC, 모니터링/알람, 감사로그/백업/DR, 레이트리밋/WAF
- 제품: 피드백 루프(사용자 설문/교사 리뷰), 공개 API(후순위), 플러그인 확장성

---

## 초기 마일스톤(요약)
- M0(2주): 현재 목표 달성(문제 출제/채점/DB 저장+대시보드 최소)
- M1(1~3개월): 코어 도메인/AI/샌드박스/워크플로 기반 완성
- M2(3~6개월): 개인화 추천/커리큘럼 자동화
- M3(6~12개월): 직무 역량/기업 연계
- M4(12개월+): 학제 확장/표준/글로벌
